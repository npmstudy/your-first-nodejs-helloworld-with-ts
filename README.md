# your-first-nodejs-helloworld-with-ts

## What is in this repository?

- use tsx as typescript register & executor
- use tsup as transpiler from ts to esm
- use tsd as type test runner

## Usages

install dependency

```
$ npm i
```

run test suite

```
$ npm test
```

transpile ts to esm for release

```
$ npm build
```

## Requirements

- Node.js v20
- TypeScript 5+
